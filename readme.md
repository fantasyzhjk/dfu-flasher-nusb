# Dfu flasher

Experimental Rust implementation of STM32 DFU flasher heavily based on STM32 dfu-util at https://github.com/dsigma/dfu-util/.

# Dependencies

For full list see Cargo.toml

 - structopt
 - usbapi-rs

# Works

 - [X] Reset STM32 to application mode.
